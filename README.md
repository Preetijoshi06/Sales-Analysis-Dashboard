# Sales-Analysis-Dashboard
Interactive Power BI dashboard built to analyze store sales, profit, and overall business performance.
#Project Overview

This project presents an interactive Power BI dashboard built to analyze store sales, profit, and overall business performance. The objective is to convert raw transactional data into meaningful insights that support data-driven decision-making.

The dashboard enables stakeholders to:

- Monitor sales and profit trends
- Evaluate category and regional performance
- Identify high-performing and underperforming products
- Understand overall business efficiency
# Problem Statement

Retail businesses often lack clear visibility into:

- Which products generate the most profit
- Regional performance differences
- Sales vs profitability imbalance
- Key drivers of business growth

This project solves these challenges using data visualization and DAX-based analysis.

# Dataset Description

The dataset consists of store-level transactional data, including:

- Order details
- Product categories and sub-categories
- Sales and profit values
- Customer segments
- Regional information
#Key Fields:
- Order Date
- Sales
- Profit
- Quantity
- Category / Sub-Category
- Region / State
- Customer Segment
# Data Cleaning & Preparation
- Removed duplicates and missing values
- Standardized category and region names
- Ensured correct data types (dates, numeric fields)
- Created relationships between tables 

📊 Data Modeling
- Established relationships between:
- Orders
- Products
- Customers
- Optimized data model for efficient querying and visualization
# DAX Calculations

Key measures created using DAX (Data Analysis Expressions):

- Total Sales
- Total Profit
- Profit Margin (%)
- Sales Growth
- Category-wise Contribution
- Top N Products Analysis
# Dashboard Features

🧩 Key Visualizations

- Sales & Profit Overview (KPIs)
- Sales Trend over Time
- Category-wise Sales Distribution
- Region-wise Performance
- Profit vs Sales Comparison
- Top & Bottom Performing Products
# Interactive Filters
- Date slicer
- Region filter
- Category filter
- Customer segment filter
# Key Insights
- A small number of products contribute significantly to total sales
- Some high-sales products have low profitability
-  Regional sales performance varies widely
- Certain categories drive most of the profit
- Some segments consistently underperform
# Business Recommendations
- Focus on high-margin product categories
- Improve pricing for low-margin high-demand products
- Optimize inventory for top-performing products
- Strengthen strategy in low-performing regions
- Target high-value customer segments
